#  App Monitoramento de Bem-Estar

##  Sobre o Projeto

O **App Monitoramento de Bem-Estar** é uma aplicação voltada para pessoas que moram sozinhas e podem enfrentar isolamento social ou depressão.

O objetivo é criar um sistema simples de acompanhamento emocional, permitindo registrar o humor diário e gerar alertas quando necessário.

---

## Problema

Pessoas que moram sozinhas podem:

- Sofrer com isolamento social
- Enfrentar episódios de depressão
- Ficar dias sem contato com outras pessoas
- Não buscar ajuda mesmo precisando

O projeto busca criar uma ferramenta de acompanhamento preventivo.

---

##  Público-Alvo

- Pessoas que moram sozinhas
- Adultos entre 18 e 65 anos
- Pessoas em acompanhamento psicológico
- Usuários que desejam monitorar saúde emocional

---

## Proposta de Valor

O aplicativo permitirá:

- Registro diário de humor
- Pergunta automática: “Está tudo bem hoje?”
- Histórico emocional
- Alerta caso o usuário não responda
- Contato de emergência configurável

---

## Funcionalidades (MVP)

- Cadastro de usuário
- Registro diário de humor (escala de 1 a 5)
- Registro de data
- Histórico de respostas
- Sistema de alerta por ausência de resposta

---

## Estrutura de Dados

### Possíveis dados coletados:

- id_usuario (INT - PK)
- nome (VARCHAR)
- email (VARCHAR)
- humor (INT)
- respondeu_hoje (BOOLEAN)
- data_registro (DATE)
- contato_emergencia (VARCHAR)

---

##  Possíveis Análises de Dados

- Identificação de quedas bruscas no humor
- Média semanal de bem-estar
